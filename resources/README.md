# 资料目录

按三大方向分类整理：

## 📂 结构

```
resources/
├── graphics/          # 图形学
│   ├── articles/     # 文章 (GDC/博客/论坛等)
│   ├── projects/     # 开源项目
│   └── notes/        # 学习笔记
├── game/              # 游戏开发
│   ├── articles/
│   ├── projects/
│   └── notes/
└── ai/                # AI
    ├── articles/
    ├── projects/
    └── notes/
```

## 📝 记录格式

### 文章 (articles/)
每个文章一个 `.md` 文件，包含：
- 标题
- 作者/来源
- 链接
- 发布日期
- 核心要点总结
- 个人笔记

### 项目 (projects/)
每个项目一个 `.md` 文件，包含：
- 项目名称
- 仓库链接
- 作者/组织
- 主要功能/亮点
- 技术栈
- 适用场景
- 个人笔记
