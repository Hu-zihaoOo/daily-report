# Transformer PyTorch 实现项目汇总

## 项目 1: hyunwoongko/transformer

**项目名称:** Transformer: PyTorch Implementation of "Attention Is All You Need"
**仓库链接:** https://github.com/hyunwoongko/transformer
**作者:** hyunwoongko
**主要功能:**
- 完整的 Transformer Encoder-Decoder 实现
- Multi-Head Attention、Positional Encoding
- 训练和推理代码示例
- 机器翻译任务演示

**技术栈:** PyTorch
**适用场景:**
- 学习 Transformer 内部实现
- 作为自定义模型的起点
- 理解 Attention 机制的代码级细节

**亮点:**
- 代码简洁，注释详细
- 完全从零实现，不依赖高级封装
- 易于阅读和修改
