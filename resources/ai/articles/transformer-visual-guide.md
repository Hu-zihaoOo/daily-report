# Transformer 架构图解指南

**标题:** The Illustrated Transformer
**作者:** Jay Alammar
**来源:** blog
**链接:** http://jalammar.github.io/illustrated-transformer/
**发布日期:** 2018-06-27 (2025年更新)

## 核心要点

1. **从宏观到微观的图解**
   - 完整的 Encoder-Decoder 流程图
   - 每个组件的详细可视化

2. **Self-Attention 一步步拆解**
   - 从词向量到 Query/Key/Value
   - 计算注意力分数
   - Softmax 归一化
   - 乘以 Value 得到输出

3. **Multi-Head Attention 可视化**
   - 8个注意力头如何并行工作
   - 不同头关注不同位置的直观展示

4. **残差连接和层归一化**
   - Add & Norm 模块的位置和作用
   - Layer Normalization 的公式
