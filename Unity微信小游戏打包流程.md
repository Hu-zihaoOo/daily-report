# Unity 微信小游戏打包流程

## 1. 环境准备
- 安装 Unity（推荐 2019.4.x 版本，部分 2021 版本可能有打包问题）
- 安装微信开发者工具
- 下载并安装微信小游戏官方转换插件

## 2. 项目配置
- 在 Unity 中导入微信小游戏 SDK 插件
- 导入后工具栏会出现"微信小游戏"菜单

## 3. 打包流程
1. **Unity 发布设置**：选择 WebGL 平台（但不需要直接发布 WebGL）
2. **打开转换工具**：点击菜单栏 `微信小游戏 → 转换小游戏`
3. **配置参数**：
   - 填写游戏 AppID（需在微信小程序公众平台申请）
   - 设置游戏方向（Landscape 横屏 / Portrait 竖屏）
   - 选择要打包的资源
   - 根据需要勾选"小游戏分包"
4. **执行转换**：点击"导出 WEBGL 并转换为小游戏"

## 4. 分包策略（重要）
- **主包限制**：微信小游戏主包不能超过 4MB
- **分包必要性**：
  - iOS 设备由于 JS 解析性能问题，通常需要分包以提升运行性能
  - iOS 还需要开启高性能模式
  - Android 设备 JS 解析性能较好，不分包也可以扫码预览
- **分包内容**：主要针对 C# 代码进行分包

## 5. 后续步骤
1. **导入微信开发者工具**：转换完成后，在微信开发者工具中选择导入项目
2. **测试调试**：在开发者工具中进行测试和调试
3. **性能优化**：
   - 包体大小优化
   - 内存优化
   - 渲染性能优化
   - 使用 GPU 动画分担压力
4. **发布上线**：提交审核
